What is Beanstalk?

Beanstalk is a new programming language and toolchain.


This website was created using Beanstalk.

Everything you see on this page, including the layout, styles, and even the text content was written in Beanstalk files and generated using the compiler's built-in HTML project builder.

The source code for this website can be found on Github.

Documentation

The docs for the basic language syntax are unfinished and not all features are fully implemented or stable.
